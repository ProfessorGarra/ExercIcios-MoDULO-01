Segundo EMMIT

O símbolo maior ">" sem as aspas, entre tags, criará a tag pai e a tag filho ( aninhamento).

O símbolo mais "+" sem as aspas, criará tags irmãs.

O símbolo acento grave "^" sem aspas, finaliza o aninhamento, assim podemos iniciar novas tag não aninhadas ou criar novos aninhamentos.

O símbolo parêntese "(" sem as aspas, cria um agrupamento, para que facilite colocar várias tags irmãs aninhadas de um mesmo pai.
