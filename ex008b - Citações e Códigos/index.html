<small>

Um texto usando a tag small, (como este aqui dentro do parêntese.)

A <sub>

Define um texto tipo subíndice (aparece mais abaixo e em menor tamanho)usando a tag sub.

A <sup>

Define um texto sobrescrito (aparece mais acima e em tamanho menor).

A <ins>

Define um texto inserido (geralmente equivale ao sublinhado).

A <do>

Define um texto eliminado (geralmente se mostra tachado).

A <code>

Define textos de modo monoespaçado em cada glifo(letra) (usualmente se mostra usando um font não proporcional como o Courier ) .

A <kbd>

Define uma entrada de texto a partir do teclado (igual ao anterior).

A <samp>

Define um exemplo (igual ao anterior).

A <var>

Define uma variável (usualmente em itálico).

A <pre>

Define um texto pré-formatado 

(usualmente respeita os espaços em branco)
.

A <abbr>

Define uma abreviatura.( como esta aqui)

A <address>

Define um endereço pessoal.

( navegador deve pegar a localização)

A <bdo>

Define a direção do texto (direita a esquerda o vice-versa).

A <blockquote>

Define uma seção que corresponde a uma cita de outro texto

(altera os margens, que podem ser alteradas via css com o comando text-indent: 35px; )
.

A <q>

Define uma citação de uma ou poucas palavras.( escreveu e não leu, o código comeu!)

A <cite>

Define uma citação de, por exemplo, um título de uma obra.(o livro das verdades inconvenientes)

A <dfn>

Destaca uma definição (usualmente se faz em itálico).
