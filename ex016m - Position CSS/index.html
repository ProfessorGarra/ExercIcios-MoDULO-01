Propriedade position

  • static
  • fixed
  • relative
  • absolute
  • sticky

Position:static

É o valor padrão da propriedade position de todos os elementos, significa que ele não tem uma posição especial definida, segue posicionado de acordo com o fluxo norma, um elemento após o outro.

Position:fixed

O elemento com o valor fixed será posicionado de forma relativa a janela onde o site é exibido. Sua posição é configurada usando os atributos right, left, top e bottom

Position:relative 02

O elemento com o valor relative é posicionado de forma relativa em relação a sua posição normal no fluxo (posição natural). E com essa position também podemos alterar a posição do elemento usando as propriedades: right, left, top e bottom.

Position:absolute

O elemento com o valor absolute, vai ser posicionado de forma relativa ao seu elemento pai, desde que seu elemento pai tenha a posição definida. Para configurar sua posição(os valores aceitos são, relative e sticky, fixed e absolute), usamos as propriedades: right, left, top e bottom.

position: absolute

Position:sticky

Com o valor sticky, o elemento é posicionado com base na da barra de rolagem. O elemento alterna entre a position relative e fixed, usa o relative até uma determinada posição de rolagem da página( dentro de uma mesma div pai, não entrando em outras divs) e depois fica fixo no local como no valor fixed. Podemos configurar ao mesmo tempo o top e o bottom.
